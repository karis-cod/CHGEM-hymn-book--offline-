# 📖 Offline Church Hymn Book

A modern, fast, and reliable offline hymn book application built with **React Native**, **Expo**, and **TypeScript**.

This application was created to help church members easily access hymns from their mobile devices without carrying physical hymn books. It is designed to work completely offline, ensuring uninterrupted access during church services, Bible studies, prayer meetings, and personal devotion.

---

## ✨ Overview

Many church members struggle with carrying hymn books to services or forget them at home. This project solves that problem by providing a digital hymn book experience that is:

* Fully Offline
* Fast and Lightweight
* Easy to Search
* Mobile Friendly
* Reliable During Worship Services
* Accessible Anywhere, Anytime

The goal is to improve hymn accessibility while preserving the worship experience.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/karis-cod/offline-church-hymn-book.git
```

### Navigate to Project

```bash
cd offline-church-hymn-book
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npx expo start
```
